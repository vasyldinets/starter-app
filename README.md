# starter app

To start:

$ npm install
$ npm install cordova -g
$ cordova platform add android (or 'ios', need Android SDK or xCode for iOS) 

To develop:
$ npm run dev

To build for production:
$ npm run build

To build app:
$ cordova build android (or 'ios', more information https://cordova.apache.org/docs/ru/latest/)


To lint you code:
$ npm run lint
